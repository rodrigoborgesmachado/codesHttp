HTTP - STATUS CODE

Descubra o que cada código HTTP significa

Listas de status HTTP, todas as referências e definições em um único lugar

Glossário de códigos:

INFORMACIONAL - 1xx
    100 - Continue
    101 - Protocolos De Comutação
    102 - Processamento
SUCESSO - 2xx
    200 - Ok
    201 - Criado
    202 - Aceito
    203 - Informações Não Autorizadas
    204 - Sem Conteúdo
    205 - Redefinir Conteúdo
    206 - Conteúdo Parcial
    207 - Vários Status
    208 - Já Reportado
    226 - IM Usado
REDIRECIONAMENTO - 3xx
    300 - Escolhas Múltiplas
    301 - Permanentemente Movido
    302 - Encontrado
    303 - Veja Outro
    304 - Não Modificado
    305 - Use Proxy
    306 - Redirecionado Temporariamente
    307 - Redirecionado Permanentemente
ERRO DE CLIENT - 4xx
    400 - Má Requisição
    401 - Não Autorizado
    402 - Pagamento Requirido
    403 - Proibido
    404 - Não Encontrado
    405 - Método Não Permitido
    406 - Não Aceito
    407 - Autorização De Proxy Requirida
    408 - Tempo Limite de Autenticação
    409 - Conflito
    410 - Sumido
    411 - Comprimento Obrigatório
    412 - Pré-condição Falhou
    413 - Carga Útil Muito Larga
    414 - URL Requirida É Longa
    415 - Tipo de Mídia Muito Não Suportado
    416 - Intervalo Solicitado Não Satisfatível
    417 - Expectativa Falhou
    418 - "Bule de Chá"
    421 - Solicitação Incorreta
    422 - Entidade Não Processada
    423 - Bloqueado/Trancado
    424 - Falha de Dependência
    426 - Atualização Requerida
    428 - Pré-condição Requerida
    429 - Muitas Requisições
    431 - Campos de Cabeçalho de Solicitação Muito Grandes
    444 - Conexão Fechada Sem Resposta
    451 - Indisponível por Razões Legais
    499 - Solicitação de Fechamento do Cliente
ERRO DE SERVIDOR - 5xx
    500 - Erro Interno de Servidor
    501 - Não Implementado
    502 - Gateway Incorreto
    503 - Serviço Indisponível
    504 - Gateway Espirou
    505 - Versão HTTP Não Suportada
    506 - Variante também Não Negociada
    507 - Armazenamento Insuficiente
    508 - Loop Detectado
    510 - Não Estendido
    511 - Autenticação de Internet Requesitada
    599 - Erro de Tempo Limite de Conexão de Rede
